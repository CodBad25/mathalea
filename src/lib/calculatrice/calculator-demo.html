🧮 Calculatrice Web Component

Utilisation basique

Le custom element s'initialise automatiquement quand il est injecté dans le DOM via la méthode connectedCallback().

Calculatrice standard

Aucune touche cassée

<my-calculator></my-calculator>

Avec touches cassées (attribut)

Définissez les touches cassées via l'attribut broken-keys en séparant par des virgules.

Sans 5 et multiplication

Défi : Faire 15

Sans division

Défi : Faire 24

Sans 0, + et *

Défi : Faire 36

<my-calculator broken-keys="5,*"></my-calculator>

Avec touches à utilisation limitée

Définissez les touches qui se cassent après un certain nombre d'utilisations via l'attribut limited-keys avec le format touche:limite. Le compteur se réinitialise avec le bouton C.

Touche 2 utilisable 1 fois

Défi : Faire 22 (ex: 2×11 ou 11+11)

Touche 5 utilisable 2 fois

Défi : Faire 55 (ex: 5×11)

Plusieurs touches limitées

Défi : Faire 100

<my-calculator limited-keys="2:1"></my-calculator>

Utilisation JavaScript

Vous pouvez aussi manipuler le composant en JavaScript :

// Récupérer l'élément
const calculator = document.querySelector('my-calculator');

// Lire les touches cassées
console.log(calculator.brokenKeys); // ['5', '*']

// Modifier les touches cassées dynamiquement
calculator.brokenKeys = ['/', '0'];
calculator.setAttribute('broken-keys', '/, 0');

// Lire les touches à utilisation limitée
console.log(calculator.limitedKeys); // [{ key: '2', limit: 1 }]

// Modifier les touches limitées dynamiquement
calculator.limitedKeys = [{ key: '2', limit: 1 }, { key: '5', limit: 2 }];
calculator.setAttribute('limited-keys', '2:1,5:2');

Caractéristiques

  • ✅ Enregistrement automatique du custom element
  • ✅ Initialisation via connectedCallback()
  • ✅ Nettoyage automatique via disconnectedCallback()
  • ✅ Prop broken-keys pour les défis
  • ✅ Prop limited-keys pour touches à utilisation limitée
  • ✅ Interface intuitive et responsive
  • ✅ Support des décimales et parenthèses
  • ✅ Fonction carré (x²)
  • ✅ Navigation avec flèches ← → pour déplacer le curseur
  • ✅ Support clavier complet (nombres, opérateurs, flèches, Backspace, Delete, Enter, Esc)
  • ✅ Curseur visuel (|) pour l'édition
  • ✅ Historique des calculs avec getHistory()
  • ✅ Animations fluides

Raccourcis clavier

  • 0-9 : Saisir un nombre
  • + - * / : Opérateurs
  • ( ) : Parenthèses
  • . ou , : Point décimal
  • ← → : Déplacer le curseur
  • Backspace : Supprimer le caractère avant le curseur
  • Delete : Supprimer le caractère après le curseur
  • Enter ou = : Calculer le résultat
  • Esc ou C : Effacer
