# Utilisation avec Anki

## Import automatisé avec AnkiConnect

1. Installer Anki depuis le site officiel d'Anki.
2. Installer l'extension AnkiConnect.
3. Lancer Anki.
4. Choisir les exercices dans MathALÉA.
5. Utiliser l'icône Anki pour importer les cartes.

Lors de la première utilisation, Anki demande l'autorisation d'utiliser AnkiConnect. Accepter pour permettre l'import.
