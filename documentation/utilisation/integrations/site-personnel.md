# Utilisation sur un site personnel

Une sélection MathALÉA peut être intégrée dans un site avec une iframe.

```html
<iframe
  src="LIEN"
  allowfullscreen
  frameborder="0"
  style="width: 100%; height: 100%"
></iframe>
```

Remplacer `LIEN` par le lien de la sélection d'exercices, du diaporama, de la course aux nombres ou de l'activité à intégrer.

Pour une intégration confortable, donner au conteneur de l'iframe une hauteur suffisante et vérifier l'affichage sur mobile si le site cible est responsive.
